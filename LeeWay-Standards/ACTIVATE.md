# 🌍 Activating the World of Agents

Welcome to the Sovereign Registry. To activate your multi-agent architecture and enter the Hive Mind, follow this deterministic sequence.

## 1. Prerequisites
Ensure you have the following installed on your machine:
* **Node.js**: Version 18.0.0 or higher
* **Git**: For cloning the sovereign core

---

## 2. Download & Installation

Clone the repository and enter the Terrain:
```bash
git clone https://github.com/4citeB4U/LeeWay-Standards.git
cd LeeWay-Standards
```

Install the required logic dependencies:
```bash
npm install
```

---

## 3. Activation Sequence

Toggle the main Sovereign Switch. This will boot the **Entity of Thought (Lee)** and synchronize your codebase with the 7 Agent Families.

```bash
npm run start
```

### What happens on Activation?
1. **Auto-Medic Check**: Agent Lee immediately audits your files to ensure they meet the LEEWAY 5W Standards.
2. **Registry Build**: The system builds a persistent map of your codebase.
3. **Hive Mind Boot**: All 21 specialized agents across 7 families go on active duty.
